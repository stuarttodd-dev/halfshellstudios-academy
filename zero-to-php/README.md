# Zero to PHP

This folder holds **hands-on PHP exercises** you can open, run, and change locally. Each subfolder is a small, self-contained project.

## Projects

### [Composer `composer.json` autoload](composer-json-autoload/README.md)

A minimal setup for **PSR-4 autoloading** with Composer: `composer.json` maps the `App\` namespace to `src/`, and `public/index.php` loads `vendor/autoload.php` and uses a namespaced class.

**Run it** (from `composer-json-autoload/`):

1. `composer install`
2. `php public/index.php`
