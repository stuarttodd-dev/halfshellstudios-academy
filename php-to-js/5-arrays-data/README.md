# Chapter 5 — Arrays & Data Transformation

This chapter covers JavaScript's core array methods — `map`, `filter`, `reduce`, `sort`, `find`, `some`, and `every` — and how they compare to PHP's array functions.

## Files

| Path | What it shows |
|---|---|
| `chapter-project/index.js` | Full data-pipeline example: filter, map, reduce, groupBy, top-N, monthly breakdown |
| `snippets/map-filter-reduce.js` | Core array methods with PHP comparison comments |
| `snippets/immutable-sort.js` | Why `sort()` mutates and how to make safe copies |

## Run the chapter project

```bash
node chapter-project/index.js
```

No dependencies — plain Node.js (v18+).
